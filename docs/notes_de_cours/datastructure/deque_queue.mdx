# üìö Deque, Queue

## Introduction

Les **deque** (double-ended queue) et **queue** sont des structures de donn√©es
lin√©aires qui permettent d'ajouter et de retirer des √©l√©ments de mani√®re
efficace.

- **Queue** : Structure FIFO (First In, First Out) - premier entr√©, premier
  sorti
- **Deque** : Permet l'ajout et la suppression aux deux extr√©mit√©s

## Queue (File d'attente)

### Import et cr√©ation
```python
from queue import Queue

# Cr√©er une queue vide
ma_queue = Queue()

# Cr√©er une queue avec une taille maximale
ma_queue_limitee = Queue(maxsize=5)
```

### M√©thodes principales

#### `put()` - Ajouter un √©l√©ment
```python
ma_queue = Queue()
ma_queue.put("premier")
ma_queue.put("deuxi√®me")
ma_queue.put("troisi√®me")
```

#### `get()` - Retirer un √©l√©ment
```python
# Retire et retourne le premier √©l√©ment (FIFO)
element = ma_queue.get()
print(element)  # "premier"

element = ma_queue.get()
print(element)  # "deuxi√®me"
```

### Exemple complet - Queue
```python
from queue import Queue

# Simulation d'une file d'attente de clients
file_clients = Queue()

# Ajouter des clients
file_clients.put("Alice")
file_clients.put("Bob")
file_clients.put("Charlie")

# Servir les clients (FIFO)
while not file_clients.empty():
    client = file_clients.get()
    print(f"Service pour: {client}")

# Sortie:
# Service pour: Alice
# Service pour: Bob
# Service pour: Charlie
```

## Deque (Double-ended Queue)

### Import et cr√©ation
```python
from collections import deque

# Cr√©er une deque vide
ma_deque = deque()

# Cr√©er une deque avec des √©l√©ments initiaux
ma_deque = deque([1, 2, 3])

# Cr√©er une deque avec une taille maximale
ma_deque_limitee = deque(maxlen=5)
```

### M√©thodes principales

#### `append()` - Ajouter √† droite
```python
ma_deque = deque([1, 2, 3])
ma_deque.append(4)
print(ma_deque)  # deque([1, 2, 3, 4])
```

#### `appendleft()` - Ajouter √† gauche
```python
ma_deque = deque([1, 2, 3])
ma_deque.appendleft(0)
print(ma_deque)  # deque([0, 1, 2, 3])
```

#### `pop()` - Retirer de la droite
```python
ma_deque = deque([1, 2, 3, 4])
element = ma_deque.pop()
print(element)    # 4
print(ma_deque)   # deque([1, 2, 3])
```

#### `popleft()` - Retirer de la gauche
```python
ma_deque = deque([1, 2, 3, 4])
element = ma_deque.popleft()
print(element)    # 1
print(ma_deque)   # deque([2, 3, 4])
```

### Exemple complet - Deque
```python
from collections import deque

# Simulation d'un buffer circulaire

# Ajouter des √©l√©ments
buffer.append("A")
buffer.append("B")
buffer.append("C")
print(buffer)  # deque(['A', 'B', 'C'], maxlen=3)

# Ajouter un 4e √©l√©ment (le premier est automatiquement retir√©)
buffer.append("D")
print(buffer)  # deque(['B', 'C', 'D'], maxlen=3)

# Utilisation comme pile (LIFO) - ajout/retrait √† droite
pile = deque()
pile.append(1)
pile.append(2)
pile.append(3)
print(pile.pop())  # 3 (dernier entr√©, premier sorti)

# Utilisation comme file (FIFO) - ajout √† droite, retrait √† gauche
file = deque()
file.append("premier")
file.append("deuxi√®me")
print(file.popleft())  # "premier" (premier entr√©, premier sorti)
```

## Performances

| Op√©ration | Queue | Deque |
|-----------|--------|-------|
| Ajouter | `put()` - O(1) | `append()` / `appendleft()` - O(1) |
| Retirer | `get()` - O(1) | `pop()` / `popleft()` - O(1) |
